# Configure an Artisyn Delivery Workspace — KB & Jira-enabled Agents (BA + Design)

Hands-on workshop that takes a freshly bootstrapped workspace to a fully
configured one:

- **Session 1 — Project Knowledge Base:** author project-kb, devops docs, the
  code-kb module map (`extract-code-kb`), domain-kb, and INDEX.md.
- **Session 2 — Agents:** add two specialist agents (a Business Analyst and a
  Design/wireframe agent) and wire Jira over MCP, scoped per role — BA authors
  issues, the lead drives workflow statuses.
- **Session 3 — Validate:** run the self-test, fix the common wiring gaps, mark phases.

Includes a precondition gate (refuses to run outside a bootstrapped workspace), a
baseline manifest snapshot, and a final Acceptance Check that proves which files
the workshop changed. Finishes at 14/15 — the pilot task is out of scope.

Level 01 (intermediate) · ~2.5–3h · Prereq: a bootstrapped workspace with ≥1 repo.
